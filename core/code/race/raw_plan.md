# 现有股票系统的增强
现有的系统，从根本上说，还是人机互动，我提要求，爱因斯坦根据要求查询，分析，提供反馈。然后我根据反馈，做修正，判断，爱因斯坦再做下一轮查询。

我想在这个基础上做一套不依赖我的思路，而让agent自由探索的系统。

# 整体思路
开多个sub agent，给他们不同的性格，投资偏好。给他们使用现有所有stock skill的权限。让这些agent 模拟炒股，每天先使用我们的工具，以及bocha分析盘面，备选，买入，卖出，每次买入写原因，卖出时分析得失。每日记录portfolio，盈亏。

爱因斯坦居中主持，每天和所有agent开会讨论，指出他们的问题，以及他们做的好的地方，写分析报告，汇总和我讨论。

# 赛马的目的
1 自发找到一些可以在A股的投资思路，而不是从我这里自上而下去寻找思路
2 赛马优秀的subagent我们可以实盘按这个思路做

# 要求
每个agent，爱因斯坦，我都要从这个流程中积累经验，而不是简单记录，没有成长
